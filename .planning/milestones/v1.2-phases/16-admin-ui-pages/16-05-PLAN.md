**Auth guard:** Copy the exact `getAdminUser()` pattern from the pricing route.

**GET handler — runs 4 parallel queries:**

1. **Booking counts:** Three separate count queries for today, this week (Monday to Sunday), this month (1st to last day):
   - Today: `.select('id', { count: 'exact', head: true }).eq('pickup_date', todayISO)`
   - This week: `.select('id', { count: 'exact', head: true }).gte('pickup_date', mondayISO).lte('pickup_date', sundayISO)`
   - This month: `.select('id', { count: 'exact', head: true }).gte('pickup_date', monthStartISO).lte('pickup_date', monthEndISO)`

2. **Monthly revenue (current + previous month):**
   - Current month: `.select('amount_czk').gte('pickup_date', monthStartISO).lte('pickup_date', monthEndISO)` — sum in JS
   - Previous month: same with previous month dates — sum in JS

3. **Revenue by vehicle class (current month):**
   - `.select('vehicle_class, amount_czk').gte('pickup_date', monthStartISO).lte('pickup_date', monthEndISO)`
   - Group by `vehicle_class` in JS: `{ vehicle_class: string, revenue: number }[]`

4. **Revenue by trip type (current month):**
   - `.select('trip_type, amount_czk').gte('pickup_date', monthStartISO).lte('pickup_date', monthEndISO)`
   - Group by `trip_type` in JS: `{ trip_type: string, revenue: number }[]`

5. **12-month revenue:**
   - `.select('pickup_date, amount_czk').gte('pickup_date', twelveMonthsAgoISO)`
   - Group by month (YYYY-MM) in JS: `{ month: string, revenue: number }[]`
   - Fill missing months with 0

interface GroupedBarChartProps {
