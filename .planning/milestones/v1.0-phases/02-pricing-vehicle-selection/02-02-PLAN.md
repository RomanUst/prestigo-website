**Calendar (react-day-picker v9 API):**

```typescript
import { DayPicker } from 'react-day-picker'

// Pickup date picker
<DayPicker
  mode="single"
  selected={pickupDate ? new Date(pickupDate + 'T00:00:00') : undefined}
  onSelect={(date) => {
    if (date) {
      const iso = `${date.getFullYear()}-${String(date.getMonth() + 1).padStart(2, '0')}-${String(date.getDate()).padStart(2, '0')}`
      setPickupDate(iso)
      // Clear returnDate if it's now before the new pickupDate
      if (returnDate && returnDate <= iso) setReturnDate(null)
    } else {
      setPickupDate(null)
    }
  }}
  disabled={{ before: new Date() }}
  modifiersStyles={{
    selected: { background: 'var(--copper)', color: 'var(--anthracite)', borderRadius: 0 },
    disabled: { color: 'var(--warmgrey)', cursor: 'not-allowed' },
    today: { outline: '1px solid var(--anthracite-light)' },
  }}
  style={{ fontFamily: 'var(--font-montserrat)' }}
  styles={{
    months: { color: 'var(--offwhite)' },
    head_cell: { color: 'var(--warmgrey)', fontSize: 13, fontWeight: 400 },
    day: { color: 'var(--offwhite)', fontSize: 13, width: 44, height: 44 },
    caption_label: { color: 'var(--offwhite)', fontSize: 13, fontWeight: 400 },
    nav_button: { color: 'var(--warmgrey)', border: '1px solid var(--anthracite-light)' },
  }}
/>
```

Note: react-day-picker v9 `styles` prop uses camelCase keys matching internal class names. If v9 changed the prop names, read the actual types from node_modules and adapt. The key visual requirements are:
- Day cells 44x44px on mobile, 40x40px on desktop
- Selected day: copper bg, anthracite text, no border-radius (flat)
- Past dates: warmgrey, not-allowed cursor
- Today: subtle anthracite-light outline
- Month navigation arrows get `aria-label="Previous month"` / `aria-label="Next month"` (react-day-picker may do this by default — verify)

**Time slot list:**

Generate 96 slots statically:
```typescript
const TIME_SLOTS: string[] = Array.from({ length: 96 }, (_, i) => {
  const h = Math.floor(i / 4).toString().padStart(2, '0')
  const m = ((i % 4) * 15).toString().padStart(2, '0')
  return `${h}:${m}`
})
```

Render as `<ul role="listbox" aria-label="Pickup time">` with each slot as `<li role="option" aria-selected={isSelected}>`.
