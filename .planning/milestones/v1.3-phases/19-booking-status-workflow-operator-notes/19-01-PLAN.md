```typescript
const bookingPatchSchema = z.object({
  id: z.string().uuid(),
  status: z.enum(['pending', 'confirmed', 'completed', 'cancelled']).optional(),
  operator_notes: z.string().max(2000).optional(),
}).refine(d => d.status !== undefined || d.operator_notes !== undefined, {
  message: 'At least one of status or operator_notes must be provided',
})
```

   - Parse `await request.json()` with `bookingPatchSchema.safeParse(body)` — return 400 with `{ error: 'Invalid payload', issues: parsed.error.issues }` on failure
   - If `parsed.data.status` is defined:
     - Fetch current booking: `supabase.from('bookings').select('status').eq('id', parsed.data.id).single()`

     - Check `VALID_TRANSITIONS[current.status]` includes `parsed.data.status` — if not, return 422 with `{ error: "Cannot transition from '${current.status}' to '${parsed.data.status}'" }`
   - Build `updatePayload` object with only the provided fields (status and/or operator_notes)
   - Execute `supabase.from('bookings').update(updatePayload).eq('id', parsed.data.id)`
   - If DB error, return 500 with `{ error: 'DB update failed' }`
   - Return 200 with `{ ok: true }`
