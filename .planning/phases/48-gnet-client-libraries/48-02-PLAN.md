## STRIDE Threat Register (ASVS L1)

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-48-08 | Information Disclosure | GnetClientError.message on API_ERROR / NETWORK_ERROR | mitigate | Test 5 asserts message contains the HTTP status but NOT the token literal 'secret-tok-xyz'; implementation interpolates only `res.status` and a static prefix |
| T-48-09 | Spoofing / Auth | Stale Upstash-cached JWT producing 401 | mitigate | postOnce returns the Response; pushGnetStatus checks `res.status === 401` and force-refreshes via getGnetToken(true); subsequent 401 yields AUTH_FAILED rather than silent failure |
| T-48-10 | DoS | Infinite retry loop on persistent 401 | mitigate | No while-loop, no recursion â€” exactly two sequential awaits to postOnce; acceptance criterion grep -v "while (" enforces this |
| T-48-11 | Tampering | Path injection via gnetResNo or griddID into URL | mitigate | Both values are passed through `encodeURIComponent` in the URL template; body uses JSON.stringify |
| T-48-12 | Repudiation | No correlation ID for outbound calls | accept | GNet's resNo serves as the natural correlation key; structured logging deferred to consumer phase |
| T-48-13 | Elevation of Privilege | Status mapper accepting arbitrary strings into GnetStatus | mitigate | prestigoToGnetStatus returns `null` for unmapped inputs; pushGnetStatus typing prevents callers from passing raw strings without explicit narrowing; phase 52 caller MUST handle null per D-01 |
| T-48-14 | Tampering | Trailing-newline pollution in GNET_API_URL / GNET_GRIDDID | mitigate | All env reads use `.replace(/\\n$/, '').trim()` |
| T-48-15 | Information Disclosure | GnetTokenError wrapped, hiding the original code | mitigate | `if (err instanceof GnetTokenError) throw err` rethrows unchanged so callers can branch on err.code |

**Block-on:** T-48-08 (token leakage in messages), T-48-10 (retry-loop DoS), T-48-15 (token error propagation). All three are covered by acceptance criteria + tests; the plan does not ship until they are green.
