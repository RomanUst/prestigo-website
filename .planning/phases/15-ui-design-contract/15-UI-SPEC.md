| Role | Token | Hex | Usage |
|------|-------|-----|-------|
| Dominant (60%) | `var(--anthracite)` | `#1C1C1E` | Page background, sidebar background |

### Accent Reserved For

Copper `var(--copper)` is used ONLY for:
- Active nav item left border (3px solid)
- KPICard label text
- Section label text (uppercase category headers)
- Save / primary CTA button border
- FilterChip active border
- StatsChart primary bar series fill
- Focus-visible outline on all interactive elements
- Booking reference text in BookingsTable

**Never:** copper as a card background, page surface, or text on `var(--anthracite-mid)` without contrast verification.

## Anti-Patterns (Enforced)

| Anti-pattern | Why | Signal to reject |
|-------------|-----|-----------------|
| White/light backgrounds | Admin shell is dark ‚Äî no white surfaces | Any `#fff`, `#f8f9fa`, `bg-white` |
| Copper as background fill | Accent only ‚Äî overuse kills exclusivity | `backgroundColor: 'var(--copper)'` on cards or sections |
| `border-radius > 4px` | Luxury geometry is sharp | Any `border-radius: 8px` or higher |
| Tailwind class names on admin components | Existing admin uses inline `style={{}}` | `className="bg-anthracite"` etc. ‚Äî use `style={{}}` |
| Booking status column | Schema has no status field ‚Äî BOOKINGS-V2-01 deferred | Any `booking.status` reference |
| Toast/notification library | Admin is single user ‚Äî inline status text is sufficient | `import toast from` any library |
| Mobile breakpoints below 1024px | Admin is desktop-only | `@media (max-width: 768px)` in admin components |
| `next/link` skipped | Phase 16 must upgrade `<a>` ‚Üí `next/link` in AdminSidebar | Plain `<a href>` in new components |
| `font-weight: 600+` on Cormorant | Destroys luxury lightness | Bold Cormorant in any admin heading |
| CZK vs EUR confusion | `amount_czk` column is CZK; display as CZK | Showing EUR for stored booking amounts |
| Font sizes outside 11px / 13px / 28px / 32px | Typography contract allows exactly 4 sizes | Any `fontSize` value not in {11, 13, 28, 32}px |
| Non-multiple-of-4 spacing in new components | Spacing scale is 4px-based | Any padding/margin/gap not in {4, 8, 16, 24, 32, 48, 64}px (legacy AdminSidebar nav padding is the sole documented exception) |
| Generic button labels | Copywriting contract requires verb + noun | `"Cancel"` as a standalone label ‚Äî use `"STOP DRAWING"` or `"Discard zone"` |
