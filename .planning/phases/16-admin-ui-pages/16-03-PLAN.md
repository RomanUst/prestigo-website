**Imports:**
```tsx
'use client'
import { APIProvider, Map, useMap } from '@vis.gl/react-google-maps'

import { TerraDrawGoogleMapsAdapter } from 'terra-draw-google-maps-adapter'
```

**Types:**
```tsx
interface Zone {
  id: string
  name: string
  geojson: { type: 'Feature'; geometry: { type: 'Polygon'; coordinates: number[][][] }; properties: Record<string, unknown> | null }
  active: boolean
  created_at: string
}
```

**Component structure — two parts:**
1. An outer component that wraps everything in `<APIProvider>` + `<Map>` and renders the zone list panel
2. A child `DrawLayer` component inside the `<Map>` that uses `useMap()` to access the map instance and initialize terra-draw

**Part 1 — ZoneMapInner (outer):**
- State: `zones: Zone[]`, `isDrawing: boolean`, `pendingPolygon: number[][][] | null`, `zoneName: string`, `deleteConfirm: string | null` (zone id being confirmed)
- On mount: fetch `GET /api/admin/zones` to populate zones
- Renders a flex row with `gap: '24px'`:
  - Left: `<APIProvider apiKey={process.env.NEXT_PUBLIC_GOOGLE_MAPS_KEY!}>` wrapping `<Map>` with the `DrawLayer` child
  - Right: Zone list panel (280px fixed width)

**Map configuration:**
```tsx
<Map
  defaultCenter={{ lat: 50.0755, lng: 14.4378 }}
  defaultZoom={11}
  mapId={process.env.NEXT_PUBLIC_GOOGLE_MAPS_MAP_ID || undefined}
  style={{ width: '100%', height: '100%' }}
  disableDefaultUI
/>
```
Map container div: `flex: 1`, `minHeight: '480px'`, `border: '1px solid var(--anthracite-light)'`, `borderRadius: '4px'`, `overflow: 'hidden'`, `position: 'relative'`.

**Draw Toolbar (positioned absolute top-right of map container):**

- When NOT drawing: Show "DRAW ZONE" button — ghost style: `border: '1px solid var(--anthracite-light)'`, `background: 'var(--anthracite)'`, `color: 'var(--warmgrey)'`, `fontFamily: 'var(--font-montserrat)'`, `fontSize: '11px'`, `textTransform: 'uppercase'`, `padding: '8px 16px'`, `borderRadius: '2px'`, `cursor: 'pointer'`
- When drawing: Show "STOP DRAWING" button — same style but `borderColor: 'var(--copper)'`, `color: 'var(--offwhite)'`
- "DRAW ZONE" click: sets `isDrawing = true` and calls `drawRef.current.setMode('polygon')` via a callback
- "STOP DRAWING" click: sets `isDrawing = false`, calls `drawRef.current.setMode('static')` or equivalent

**Part 2 — DrawLayer child component (inside <Map>):**
Uses `useMap()` hook. In a `useEffect` on map:

1. **Fix div ID bug:** `const mapDiv = map.getDiv(); if (!mapDiv.id) mapDiv.id = 'terra-draw-map-container';`

2. **Initialize terra-draw:**
```tsx
const draw = new TerraDraw({
  adapter: new TerraDrawGoogleMapsAdapter({ lib: google.maps, map, coordinatePrecision: 9 }),
  modes: [new TerraDrawPolygonMode(), new TerraDrawSelectMode({ flags: { polygon: { feature: {} } } })],
})
```

3. **MANDATORY: Wait for `ready` event before calling `setMode`:**
```tsx
draw.on('ready', () => { drawRef.current = draw })
```

4. **Listen for `finish` event** to capture completed polygon:
```tsx
draw.on('finish', (id: string) => {
  const snapshot = draw.getSnapshot()
  const feature = snapshot.find(f => f.id === id)
  if (feature && feature.geometry.type === 'Polygon') {
    onPolygonComplete(feature.geometry.coordinates)
    draw.clear()
  }
})
```

5. `draw.start()` — and return `() => { draw.stop() }` in cleanup

```tsx
import dynamic from 'next/dynamic'

  ssr: false,
  loading: () => (
    <div style={{
      background: 'var(--anthracite)',
      minHeight: '480px',
      borderRadius: '4px',
      display: 'flex',
      alignItems: 'center',
      justifyContent: 'center',

        fontSize: '13px',
        color: 'var(--warmgrey)',

        Loading map...
      </span>

})
```
